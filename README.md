# 专注时间

> 专注时间iOS端，提供专注时间倒计时，专注时间设置，专注时间统计、离开提醒等功能，旨在于帮助人们管理和专注于时间。

## Features

- [x]  倒计时
- [ ] 完成和退出的提醒
- [x] 专注设置
- [x] 个人中心
- [ ] 倒计时
- [ ] 番茄逻辑
- [ ] 休息时长统计
- [ ] 换肤功能
